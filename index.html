用于生成符合 HTX 品牌规范的长图、海报和运营视觉物料，让版式、色彩、字号与 Logo 使用保持一致。

统一、清晰、可复用

这份 Skill 将 HTX 的品牌视觉语言转化为可执行规范，适用于活动内容、权益说明、数据对比、交易赛事与其他运营物料。

品牌主题

支持 brand-blue 与 vip-gold 两套主题，统一渐变方向、品牌色和内容层级。

可读性规范

以画布比例推导字号，同时设置正文最小字号，兼顾跨尺寸适配与实际阅读体验。

HTML + CSS

使用标准 HTML 盒模型排版长图内容，并附带自适应缩放逻辑，减少裁切和溢出。
